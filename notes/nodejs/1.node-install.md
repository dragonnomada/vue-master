# Node JS Installation

[https://nodejs.org/](https://nodejs.org/en/)

## Download NodeJS

> curl -o node-v16.msi "https://nodejs.org/dist/v16.13.0/node-v16.13.0-x64.msi"

## Install Node JS

> msiexec /i node-v16.msi

## Cleanup Installer

> rm node-v16.msi
