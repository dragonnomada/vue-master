## CONFIGURE SERVER

> Create folder `tickets-server`

> Generate `package.json`

	cd tickets-server
	npm init -y

> Dependencies

	npm install --save express
	npm install --save mysql2

> Ignore Git `.gitignore`

dist
package-lock.json
